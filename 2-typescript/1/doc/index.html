TypeScript

https://www.typescriptlang.org/

The goal is to makes js a full language with typing/class/abstract, independant of ECMAScript version.

This allow to cover past and future release of ECMAScript.

Install:

npm i -g typescript

Do it by 'hand'

tsc src/app.ts
node src/app.js

Initiate a tsconfig.json

tsc -init

Example:

{
    "compilerOptions": {
        "module": "commonjs",
        "target": "es5",
        "noImplicitAny": false,
        "sourceMap": true,
        "outDir": "./dist"
    },
    "include": [
        "./src/**/*"
    ]
}

Result:

> tsc
> node dist/app.js
[ { name: 'Joe', age: 42 } ]
end
